  • git clone (path to repository) - клонування репозиторію в локальну директорію
  • git status - перевірка статусу поточного локального репозиторію
  • git add (path to file) - додавання нового файлу для відслідковування змін git'ом
  • git commit -am "added new file index.html" - фіксація змін і коментування їх
  • git push - відправлення всіх заcommitчених змін/файлів в репозиторій на сервер
  • git pull - забрати "свіжі" зміни/оновлення з сервера в локальний репозиторій
